# 🌐 DevKitX — Essential Tools for Designers & Developers

DevKitX is a modern, high-performance landing page built for showcasing browser extensions designed for UI/UX designers, developers, and creators.

It features smooth animations, teleporting icons, custom GSAP effects, and an interactive download experience.

## ✨ Features

### 🎨 Beautiful Hero Section
- Smooth fade-in animations  
- Bold, crisp typography  
- Highlight text with clean UI layout  

### 🌀 Interactive Icon Teleport Animation
A continuous teleport-style shuffle:
- Icons shrink to center  
- Reappear randomly  
- Repeats endlessly  
- Stops when user scrolls  

### ⚡ GSAP-Powered Animations
- Intro fade-in animation  
- Scroll-triggered animations  
- Particle explosion on download button  

### 📥 Download Button with FX
A button that:
- Animates on click  
- Triggers particle explosion  
- Downloads a real file  

### 🚀 Smooth Scrolling with Lenis
Provides buttery-smooth scrolling effects.
